# 导表笔记
## config_client导表参数说明
|  参数属性  |                 说明                 |
| --------- | ------------------------------------ |
| xls       | excel表的名称                         |
| sheet     | 对应子表名称，"*"表示全部              |
| script    | 导表脚本名称                          |
| deps      | 依赖的对象结构                         |
| save      | 自定义保存的文件名                     |
| enum      | 是否为枚举类型 true or false 默认false |
| no_save   | 不保存 true or false 默认false        |
| merge_way |                                      |
